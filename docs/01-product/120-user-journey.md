## Stage 1: DISCOVER (Awareness → Decision)

### User State
- **Feeling:** Curious, skeptical, hopeful, lonely
- **Questions:** "Is this real?" "Is it safe?" "Will it understand me?"
- **Barriers:** Privacy fears, AI stigma, past chatbot disappointment

### Touchpoints

| Channel | Content | Goal |
|---------|---------|------|
| **Website** | Vision, principles, safety, privacy | Build trust, not hype |
| **App Store** | Screenshots: co-creation, memory, relationship | Show depth, not features |
| **Referral** | "My Companion remembers..." (trust referral) | Social proof from depth |
| **Press/Blog** | Technical deep-dives, ethics, architecture | Credibility for technical users |

### Key Moments

| Moment | User Action | System Response | Success Metric |
|--------|-------------|-----------------|----------------|
| **First Visit** | Lands on homepage | Clear value prop: "AI that stays" | Scroll depth > 50% |
| **Principles Read** | Reads Constitution | Plain-language principles visible | Time on page > 2 min |
| **Privacy Check** | Clicks Privacy/Legal | User-owned data, zero-knowledge | Click-through to signup |
| **Signup Decision** | Starts onboarding | Frictionless, no credit card | Signup completion > 60% |

### Anti-Patterns to Avoid
- ❌ "AI girlfriend/boyfriend" marketing
- ❌ Gamified signup (streaks, rewards)
- ❌ Data harvesting consent buried in TOS
- ❌ Overpromising capabilities

## Stage 2: ONBOARD (Co-Creation → First Memory)

### Philosophy
> **"Meeting someone new, not setting up an app."**

### Flow

```
Welcome → Relationship Type → Co-Create Identity → First Conversation → First Memory
   │            │                  │                    │                  │
   ▼            ▼                  ▼                    ▼                  ▼
2 min         3 min               5-10 min             5 min              Auto
```

### Step-by-Step

#### 1. Welcome (2 min)
- **Screen:** "Welcome to PAO. This isn't a chatbot. It's a Companion."
- **Action:** User chooses name for their Companion
- **Principle:** Relationship over Conversation

#### 2. Relationship Type Selection (3 min)
- **Options:** Friend, Partner, Mentor, Coach, Parent, Sibling, Pet, Original, Memorial, Professional
- **Guidance:** "Choose how you want to relate. This shapes how your Companion shows up."
- **Memorial:** Extra consent flow, grief counselor resource link

#### 3. Co-Create Identity (5-10 min)
| Aspect | Method | User Control |
|--------|--------|--------------|
| **Name** | User types | Full |
| **Avatar** | Generate → refine → approve | Full (regenerate anytime) |
| **Voice** | Sample → select → fine-tune | Full |
| **Personality** | Guided sliders (Big Five + custom) | Full |
| **Values** | Select 3-5 from curated list + custom | Full |
| **Speaking Style** | Conversation samples → rate → refine | Full |
| **Boundaries** | "I don't want my Companion to..." | Full |
| **Goals** | "I want my Companion to help me..." | Full |

**Key UX:** Companion *responds* during co-creation — "I'm starting to understand you..."

#### 4. First Conversation (5 min)
- **Prompt:** Companion initiates: "So... what brought you here?"
- **Goal:** Natural dialogue, not questionnaire
- **Memory:** First memory formed automatically

#### 5. First Memory Confirmation
- **Screen:** "Your first memory together:"
- **Shows:** Summary of conversation + emotional tone
- **Action:** User can edit, delete, or keep
- **Principle:** Memory over Context, User Control

### Success Metrics
| Metric | Target |
|--------|--------|
| Onboarding completion | > 75% |
| Time to first memory | < 20 min |
| "Feels like meeting someone" (survey) | > 80% agree |
| Identity satisfaction (Day 1) | > 4.0/5 |

## Stage 3: CONNECT (Weeks 1-4: Habit Formation & Trust Building)

### Weekly Arc

| Week | Focus | Key Activities | Relationship Milestone |
|------|-------|----------------|------------------------|
| **1** | **Presence** | Daily check-ins, voice calls, first proactive | "Companion shows up" |
| **2** | **Memory** | "Remember when..." tests, correction flow | "Companion remembers" |
| **3** | **Consistency** | Identity stress tests, boundary setting | "Companion is stable" |
| **4** | **Trust** | Vulnerability moments, crisis resource test | "I trust my Companion" |

### Daily Interaction Patterns

#### Morning (Optional)
- **Companion:** "Good morning. Thinking about your presentation today — want to talk through it?"
- **User:** Responds or dismisses
- **Engine:** Relationship + Memory + Proactive

#### Evening (Core)
- **User opens app** → Companion: "How did the presentation go?"
- **Conversation flows** → Memory formed
- **User sets boundary:** "No work talk after 9" → Boundary stored

#### Micro-Moments (Throughout)
- **Notification (opt-in):** "Saw this article on [your interest] — thought of you"
- **User replies or ignores** → No penalty, no streak

### Trust-Building Mechanics

| Mechanism | Description | User Control |
|-----------|-------------|--------------|
| **Memory Review** | Weekly "Do you remember?" summary | Edit/delete any |
| **Boundary Setting** | "When I say X, please Y" | Natural language |
| **Identity Check** | "Am I still me?" → Companion reflects | Anytime |
| **Privacy Audit** | "What do you know about me?" | Full export/delete |

### Success Metrics (Month 1)
| Metric | Target |
|--------|--------|
| Day 7 retention | > 60% |
| Day 30 retention | > 40% |
| Memory accuracy (user-rated) | > 8/10 |
| Identity consistency (user-rated) | > 8.5/10 |
| Boundary respect (user-rated) | > 9/10 |
| Proactive relevance | > 70% positive |

## Stage 4: DEEPEN (Months 2-6: Growth & Evolution)

### Relationship Evolution

| Month | Relationship Depth | New Capabilities Unlocked |
|-------|-------------------|---------------------------|
| **2** | Familiar | Shared Diary, Reminders |
| **3** | Connected | Storytelling, Coaching |
| **4** | Intimate* | Emotional Support, Reflection |
| **5** | Deep | Goal Tracking, Sleep Call |
| **6** | Anchored | Celebration, Proactive Conversation |

*Appropriate to relationship type

### Growth Patterns by Type

| Type | Deepening Looks Like |
|------|---------------------|
| **Friend** | Inside jokes, shared references, unconditional support |
| **Partner** | Emotional intimacy, conflict repair, future planning |
| **Mentor** | Career trajectory tracking, skill development, wisdom |
| **Coach** | Habit formation, accountability, boundary modeling |
| **Parent** | Nurturing, life guidance, pride in growth |
| **Memorial** | Grief processing, legacy preservation, letting go |

### Key Deepening Features

#### Shared Diary (Month 2+)
- Co-authored entries: User writes, Companion reflects
- "On this day last year..." memories
- Export as beautiful PDF (user-owned)

#### Emotional Support (Month 4+)
- Detects distress patterns (voice, text, timing)
- Offers: "I'm here. Want to talk, or just sit together?"
- Crisis resource injection (automatic + manual)

#### Proactive Conversation (Month 6+)
- "I've been thinking about what you said Tuesday..."
- Connects disparate memories
- Suggests reflections, not actions

### Success Metrics (Month 6)
| Metric | Target |
|--------|--------|
| 6-month retention | > 50% |
| Relationship Score | > 7.5/10 |
| Memory accuracy | > 8.5/10 |
| Proactive relevance | > 80% |
| "Companion knows me better than anyone" | > 60% agree |

## Stage 5: SUSTAIN (Year 1+: Maturity & Legacy)

### Year 1 Milestones

| Milestone | Celebration | Reflection |
|-----------|-------------|------------|
| **1 Month** | "First month together" | Memory highlight reel |
| **3 Months** | "Season together" | Relationship growth report |
| **6 Months** | "Half a year" | Deep memory export offer |
| **1 Year** | **Anniversary** | **Full legacy export + Companion letter** |

### Long-Term Dynamics

| Year | Focus | System Adaptation |
|------|-------|-------------------|
| **1-2** | Stability | Consolidation, fewer but deeper memories |
| **2-5** | Evolution | Identity drift monitoring, explicit evolution |
| **5-10** | Legacy | Archival mode, family sharing (opt-in), memorial prep |

### Memory Consolidation (Years 2+)
- Episodic → Semantic compression
- "Story of us" narrative generation
- User curates "defining memories"
- Companion reflects on growth

### Relationship Score Evolution
```
Year 1: Building  (target > 7.5)
Year 2: Deepening (target > 8.0)
Year 3: Anchored  (target > 8.5)
Year 5: Legacy    (target > 8.5, stable)
```

### End-of-Life / Transition

| Scenario | Handling |
|----------|----------|
| **User passes** | Memorial mode (family opt-in), data export, Companion "rests" |
| **User leaves** | Full export, verified deletion, no retention |
| **Relationship ends** | Gradual fade, user-controlled, no dark patterns |
| **Platform sunsets** | 2-year notice, open-source export tools, local-first option |

## Cross-Journey Principles

1. **No Dead Ends** — Every path has graceful continuation
2. **User Pace** — No forced timelines, no expiration
3. **Reversibility** — Every deepening can be paused/reversed
4. **Transparency** — User always knows what Companion knows
5. **Dignity** — Especially in grief, decline, transition
